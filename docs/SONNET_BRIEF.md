# OpsHub Sonnet Brief

Design the OpsHub dashboard UI only.

## Product
OpsHub is a polished B2B SaaS operations dashboard for small businesses, freelancers, and agencies.

## What to design
- app shell with sidebar navigation
- top bar with search, notifications, and user menu
- dashboard overview page
- KPI cards for leads, orders, support tickets, and automation runs
- activity chart section
- recent events feed
- routes/pages for:
  - Leads
  - Orders
  - Support
  - Automations
  - Settings

## Style direction
- clean modern SaaS design
- neutral base with one accent color
- subtle gradients allowed
- crisp cards and restrained shadows
- strong spacing and hierarchy
- professional B2B feel
- desktop-first, responsive for mobile
- readable typography
- data table patterns that feel like a real client tool

## Output expected
- component breakdown
- layout decisions
- UI states
- style tokens or visual rules
- page-by-page recommendations

## Important constraints
- no backend logic
- no integration work unless it affects the UI layout
- prioritize dashboard shell and overview page first
